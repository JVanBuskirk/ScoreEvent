PMask, a CMask implementation in Python

What is CMask?

CMask is a stochastic event generator for Csound written by Andre Bartezki.

I would suggest you to read the CMask manual as it remains a central reading to understand the concepts behind CMask and PMask.

Why PMask?

PMask is a complete reimplementation of CMask using the Python scripting language. Although it is possible to write, in Python and using PMask a parser for the CMask's original input files, PMask is designed as a set of classes to be used directly inside Python script.

Of course you have to know how to write simple programs in Python.

How does PMask code look like?

This is the PMask version of the first CMask example from the CMask manual:
